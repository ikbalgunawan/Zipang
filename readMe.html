    Zipang
    • wahdania.prasetyo

    Zipang webapps adalah sebuah platform untuk mengakomodir keperluan para agents yang terdaftar dalam "Zipang X Veesion", sehingga para agents dapat melihat progres pekerjaan, honorarium secara real-time, maupun melakukan print-out slip honorarium setiap bulannya.

    Mulai

    Agent dapat melakukan proses pendaftaran dengan meng-klik tombol pada halaman awal Agent akan dihadapkan pada formulir pendaftaran yang harus diisi sesuai dengan data asli seakurat mungkin Format username harus sesuai dengan format username pada aplikasi veesion tanpa menggunakan awalan "Zepang_" (contoh: jika pada aplikasi veesion username agent adalah "Zepang_Budi", maka pada proses pendaftaran username yang harus diisi hanyalah "Budi").

    Setelah formulir sudah diisi, agent dapat melajutkan proses pendaftaran dengan menekan tombol "Sign Up".

    Masuk

    Agent dapat melakukan proses login pada halaman awal webapps dengan memasukan username dan password yang sudah didaftarkan, kemudian agent akan dialihkan menuju halaman "Dashboard".

    Pada halaman "Dashboard" agents dapat melihat detail pekerjaan, banyaknya invoice yang sudah dibayarkan, jumlah pendapatan yang sudah dihasilkan.

    Halaman "Dashboard" juga akan menampilkan 5 Periode terkahir dari pekerjaan agent dan 5 Invoice terakhir yang sudah terbit.



    Setiap tanggal 15 atau lebih dalam setiap bulannya, agent wajib melakukan proses print honorarium untuk dapat diproses oleh admin menuju tahap pencairan dana agent tersebut, dengan cara meng-klik tombol Untuk details slip honorarium dapat dilihat pada bagian "Latest Slip Honorarium".

    Perubahan Data Profile Agents

    Agent dapat melakukan perubahan data jika terjadi kesalahan input pada proses pendaftaran dengan cara menuju halaman edit profile yang terletak pada pojok kanan atas sesuai dengan nama agent yang bersangkutan.
